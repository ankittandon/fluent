# Fluent

Fluent is desktop software that helps anyone get instant, in-context support while using software.
With one action, Fluent captures what is on screen, listens to your question, and sends image + audio context to an LLM running either on-device or in the cloud.
The model then returns clear, step-by-step guidance for exactly what to do next in the app you are using.

## What Fluent does

- Captures visual context from your desktop when you ask for help
- Captures spoken questions through your microphone
- Sends multimodal context (image + audio) to your selected LLM backend
- Supports local/on-device inference and cloud-hosted models
- Delivers just-in-time, workflow-aware instructions without forcing users to leave their task

## Why this matters

Most users do not need more documentation. They need help in the exact moment they get stuck.
Fluent closes that gap by turning the current desktop state into actionable guidance.

## Core experience

1. You trigger Fluent while working in any desktop app.
2. Fluent captures the relevant screen context and your spoken prompt.
3. Fluent packages this context for an LLM (local or cloud).
4. Fluent returns practical next steps tailored to the UI in front of you.

## Use cases

- Software onboarding and feature discovery
- Just-in-time support for complex workflows
- Internal tool adoption for teams
- Reducing training overhead and support ticket volume
